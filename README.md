# Holofilm-pm
## Nodejs版本推荐V16+
## 使用npm
### 安装项目依赖
```
npm install
```

#### 在本地运行
```
npm run dev
```

#### 打包编译
```
npm run build
```

#### 格式化代码
```
npm run lint
```


## 使用yarn
### 安装项目依赖
```
yarn install
```

#### 在本地运行
```
yarn dev
```

#### 打包编译
```
yarn build
```

#### 格式化代码
```
yarn lint
```
